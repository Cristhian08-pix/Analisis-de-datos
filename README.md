    requerida.
    Graficos mostrados:
    - Campana de Gauss
    - Observado
    - Grafico de dispersión
    - Histograma

## 2. Reporte de pesos Regional
    La información cargada en Oracle es procesada mediante consultas SQL y definidas en Power query para su automatización.
    Se utiliza lenguaje DAX para aumentar la capacidad de analisis, esto dentro de Power BI, con un alcance a nivel regional.
    Incluyendo las plantas: México Norte, Monterrey, Puebla.
    
    Detalla el comportamiento de los diferentes modelos trabajados en la organización con la capacidad de segmentar
    la información por: Planta, periodo, tecnologia, maquina, producto. Lo cual hace un rastreo preciso de la
    variable peso para calculos de consumos de resina.
    El reporte visualiza:
    - Consumos de resina
    - Reporte estadistico con variable como: Promedio, mediana, Cp, Cpk, asi como un grafico de dispersión.
    - Resume la información de los modelos que se encuentran por arriba/debajo de los limites de especificación.
    - Realiza un comparativo entre los modelos agrupados por tecnologia/planta.
    Se definen aquellos productos por fuera de especificación a los cuales se les tiene que hacer un seguimiento continuo
    para monitorear su peso, y esto se logra con la información presentada siguiendo tendencias y comportamientos.
