# Analisis de datos
A continuación se presentan algunos de los proyectos desarrollados previamente en el ámbito de análisis de datos y automatización de procesos. Cada iniciativa refleja la aplicación de herramientas avanzadas para optimizar la gestión de información, mejorar la precisión en los resultados y reducir tiempos operativos.

## *1.Control estadistico*
Este módulo, desarrollado en Visual Basic for Applications (VBA) para Excel, **automatiza** la generación de informes estadísticos
con una capacidad de procesamiento de hasta 100,000 valores. 
### Precisión tecnica:
* Promedio y desviación estándar 
* Índices de capacidad del proceso (Cp, Cpk)
* Nivel sigma 
* Valores mínimo y máximo
* Resultados observados y estimados (% y ppm)
    
El sistema permite evaluar la información con o sin especificaciones, adaptando la ruta de análisis según la necesidad.
    
### Gráficos generados
* Campana de Gauss
* Histograma
* Gráfico de dispersión (clásico y radial) 
* Gráfico de cajas y bigotes
* Resultados observados 

En consecuencia permite un analisis estadistico **claro, completo, preciso y eficaz.**

## *2. Reporte de pesos Regional*
La información cargada en Oracle es procesada mediante consultas SQL y **automatizada** a través de Power Query.
Para ampliar la capacidad de análisis, se emplea el lenguaje DAX dentro de Power BI, logrando un alcance a nivel
regional que incluye las plantas de México Norte, Monterrey y Puebla.
### Alcance del reporte

El sistema detalla el comportamiento de los diferentes modelos trabajados en la organización, con la capacidad de segmentar la información por:
* Planta
* Periodo
* Tecnología
* Máquina
* Producto

Esto permite un rastreo preciso de la variable peso, fundamental para el cálculo de consumos de resina.
### Visualizaciones generadas

* Consumos de resina
* Reporte estadístico con métricas como: promedio, mediana, Cp, Cpk
* Gráfico de dispersión
* Resumen de modelos fuera de límites de especificación
* Comparativos entre modelos agrupados por tecnología y planta

### Seguimiento de productos

El sistema **identifica** aquellos productos que se encuentran fuera de especificación y requieren monitoreo continuo. 
Gracias a las tendencias y comportamientos presentados, se facilita la **supervisión** del peso y el control de consumos de resina.

## *3. Automatización/Cargue de información con RPA(Robotic Process Automation)*

La **automatización** mediante RPA permite el cargue masivo de información en cualquier plataforma, reduciendo significativamente
los tiempos de ejecución, eliminando errores humanos y garantizando un proceso dinámico y eficiente.

En este caso, la programación se realiza con Pulover’s Macro Creator (PMC). Para el ejemplo presentado, se diseñó un robot 
que automatiza el cargue de información en la plataforma Oracle, integrando PMC con una macro en Excel. Esta conexión incrementa 
la eficiencia del proceso y asegura una **gestión** confiable de grandes volúmenes de datos.
    
### Beneficios principales

* Reducción de tiempos operativos
* Eliminación de errores humanos en el proceso de cargue
* Automatización dinámica adaptable a diferentes plataformas
* Integración eficiente entre herramientas (PMC, Excel, Oracle)

## *4. Analisis de costos/Buenas practicas de manufactura*

### Diseño de tablero dinámico con indicadores clave de desempeño

Se ha desarrollado un tablero dinámico que integra indicadores estratégicos relacionados con costos, 
rechazos e incumplimientos a las BPM’s. El sistema permite un análisis detallado por:

* Máquina
* Tecnología
* Operador
* Periodo

Este enfoque facilita el seguimiento preciso de la merma generada como costo interno, proporcionando información 
crítica para la toma de decisiones. Con ello, la organización puede implementar acciones correctivas oportunas 
y fortalecer su proceso de mejora continua.

El tablero también ofrece un desglose específico de:

* Fallas por máquina
* Rechazos por operador
* Incumplimientos a las BPM’s

Todo ello respaldado por un sistema de rastreo eficaz y altamente eficiente, 
que asegura visibilidad integral y soporte a la gestión operativa.
