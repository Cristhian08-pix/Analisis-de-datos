## 1.Control estadistico
    Programación en Visual Basic for Aplication en Excel para la aotomatización del informe en cuestión
    Procesamiento de datos: Capacidad de 100,000
    Recuento de variables: Datos, promedio, desviación estandar, Cp, Cpk, sigmas, Valor minimo, Valor maximo, Observado(%),
    Observado(ppm), Estimado(%), Estimado(ppm).
    Ofrece la evaluación de la data con o sin especificaciones, tomando rutas distintas segun la asignación requerida.
    Graficos mostrados:
    - Campana de Gauss
    - Observado
    - Grafico de dispersión
    - Histograma
    - Grafico de dispersión radial
    - Grafico de cajas y bigotes
